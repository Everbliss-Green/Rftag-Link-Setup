# web_rftag

A new Flutter project.

## Getting Started

This project is for assigning groupid and frequencies for rftag devices using
its gonna usb serialport  as a library to connect rftag wired
https://pub.dev/packages/serial for web
and then one button to find device and then when clicked it will assign a groupid and a frequency

theres going to be a text area for groupid and theres going to be dropdown for frequency

the serial port command for group id is
rftag settings groupid set 000000
for frequency its
rftag settigns lora freq 923875000
